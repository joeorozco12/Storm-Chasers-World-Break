# Storm Chasers: World Break Roblox Store Page Package

This package turns the March 22, 2026 benchmark review into a publish-ready Roblox experience page brief for **Storm Chasers: World Break**. It is grounded in the game vision from `docs/GAME_FOUNDATION.md` and is meant to guide store copy, icon art, thumbnail production, and review.

## Positioning

- Public title: `Storm Chasers: World Break [ALPHA]`
- Core promise: Living weather changes the world in real time, and players chase the danger instead of waiting for content.
- Visual lane: Clear objective framing, oversized weather spectacle, and strong creature scale. Stay closer to readable adventure art than anime-combat branding.
- Safe benchmark takeaways:
  - clear fantasy-first hooks
  - strong icon readability at small sizes
  - thumbnail sets built around distinct play moments
  - short feature lists with concrete verbs
- Do not copy:
  - logos
  - poses
  - compositions
  - exact phrasing
  - a signature palette from another game

## Final Store Copy

- Preview-safe opening:
  `Storm Chasers: World Break is a social survival-adventure where living weather reshapes the map in real time. Forecast the sky with friends, chase rare storm fronts, survive the impact, discover what storms reveal, and build shelters that hold.`
- Core verbs:
  - `Forecast`
  - `Chase`
  - `Survive`
  - `Discover`
  - `Build`

### Copy Rules

- Lead with the fantasy hook, not patch notes.
- Keep the first paragraph focused on weather, discovery, and co-op.
- Use short action-first sentences over long FAQ-style blocks.
- Do not add reward codes, long changelogs, or maintenance notes above the hook.
- Add a short update line above the hook only after live retention features justify it.

## Visual Package

### Icon Brief

- Concept: Lightning Serpent crossing the frame above a grounded player holding a rod or anchor tool.
- Readability goal: The icon must still read as storm + danger + creature at `112x112` and `48x48`.
- Composition:
  - serpent body arcs diagonally across the frame
  - player stays secondary and readable as a grounded human silhouette
  - one bright lightning path creates the focal line
  - avoid a logo if it competes with silhouette clarity
- Palette:
  - storm-cyan
  - lightning-orange
  - cloud charcoal
  - wet steel
  - signal white
- Avoid:
  - anime battle faces
  - purple horror glitch treatments
  - meme exaggeration
  - cluttered UI overlays

### Thumbnail Order

1. `CHASE THE FRONT`
   Giant storm wall over Thunderstep Plains with multiple players running toward the same weather event. The sky must dominate the frame and clearly show why the group is moving.
2. `TAG THE SERPENT`
   Server-wide Lightning Serpent encounter at the Coil Spire. Make the creature visibly oversized relative to players and landmarks.
3. `SURVIVE THE IMPACT`
   Shelter under pressure from lightning, wind, and floodwater. Show the structure, not just the characters, so the survival fantasy is readable.
4. `FORECAST THE SKY`
   Forecast tool and journal moment before a rare storm lands. The scene should communicate planning, not passive menu browsing.
5. `DISCOVER WHAT IT REVEALS`
   Storm aftermath exposing a path, track, ruin, or rare creature clue. The weather should feel like the reason the discovery became possible.
6. `MAKE THE SERVER STOP`
   Eclipse, aurora, meteor, or other shared sky event with a visible group. The image should sell the whole-server spectacle, not a solo screenshot.

### Thumbnail Art Rules

- Prioritize one clear idea per image.
- Keep creature scale, sky drama, and map change legible before adding detail.
- Default to no text overlays; only add minimal text if an image fails the clarity test without it.
- Use cinematic contrast and atmosphere, but keep the objective readable on a mobile-sized card.
- Favor environment spectacle over combat posing.
